Complete the implementation of the remaining non-functional Settings options. Every setting listed below must perform a real function and persist across app restarts.

## 1. Resume Last Read

Implement a fully functional "Resume Last Read" setting.

Requirements:

* When enabled, automatically save:

  * Surah
  * Ayah
  * Scroll position
  * Selected translation
  * Selected Tafseer (if open)
* When the user reopens the Quran, automatically resume from the exact previous reading position.
* When disabled, always open the Quran from the beginning or the Surah list.
* Persist the user's preference.

## 2. Highlight Last Read Verse

Implement a visual indicator for the last verse the user was reading.

Requirements:

* Highlight only the saved last-read verse.
* Use a subtle premium highlight (do not overpower the theme).
* Automatically remove the highlight when the user continues reading and reaches a newer verse.
* Respect the toggle in Settings.

## 3. Daily Dua Reminder

* Notification opens directly to the Daily Reflection / Daily Dua section.
* Persist the preference.

## 4. Daily Inspiration Reminder
