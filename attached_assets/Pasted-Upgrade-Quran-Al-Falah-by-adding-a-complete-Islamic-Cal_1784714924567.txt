Upgrade Quran Al-Falah by adding a complete Islamic Calendar & Islamic Events feature.

Goal

Create a modern, elegant, and fully responsive Islamic Calendar page that becomes one of the main features of Quran Al-Falah while maintaining the existing design language and without breaking any current functionality.

1. Current Date Card

At the top of the page display:

Current Hijri Date
Current Gregorian Date
Day of Week
Islamic Month Name
Hijri Year

Example:

27 Muharram 1448 AH
Tuesday
22 July 2026

Use accurate Hijri calculations or a trusted Islamic calendar source already compatible with the project. Do not hardcode dates.

2. Beautiful Monthly Hijri Calendar

Display a complete monthly Hijri calendar.

Requirements:

Previous month button
Next month button
Highlight today's Hijri date
Show Gregorian day number inside each Hijri date cell in smaller text
Responsive on mobile and desktop
Smooth transitions when changing months
Maintain Quran Al-Falah dark theme
3. Islamic Events Section

Add a section called:

Upcoming Islamic Events

Display important dates including:

Islamic New Year
Ashura
Mawlid (12 Rabi' al-Awwal)
Isra & Mi'raj
Mid Sha'ban
Ramadan
Laylat-ul-Qadr
Eid-ul-Fitr
Day of Arafah
Eid-ul-Adha

Each event should show:

Event name
Hijri date
Gregorian equivalent
Days remaining

Example:

Ramadan begins

1 Ramadan 1448 AH

Begins in 183 days

When a user taps any event,
open a beautiful modal/page showing:

Event name
Importance
Short authentic description
Relevant Qur'an verses (if applicable)
Authentic Hadith references (if applicable)

Use only authentic information.

Muharram

"Muharram is one of the four sacred months mentioned in the Qur'an..."
