## Design Requirements

* Follow the existing Rah-e-Safia premium green theme.
* Use modern settings cards with rounded corners.
* Use meaningful icons for every setting.
* Maintain consistent spacing and typography.
* Add subtle animations for switches and navigation.
* Ensure excellent mobile responsiveness.
* Support both Light and Dark themes.
* Use expandable sections where appropriate to avoid long scrolling.
* Do not change any existing functionality unless necessary.
* Any feature not yet implemented (such as notifications) should appear as disabled or marked "Coming Soon" rather than being removed.

---

## Technical Requirements

* Persist all settings using local storage or the existing persistence system.
* Restore user preferences automatically when the app launches.
* Structure the code so additional settings can be added easily in future updates.
